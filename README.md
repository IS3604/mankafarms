# Manka Farms — Jungle Farmstay Website

Official website for **Manka Farms**, a jungle farmstay near Pench Tiger Reserve, Seoni, Madhya Pradesh.

## 🌿 About

Manka Farms is a 4-bedroom farmstay (sleeps 12) located in Amajhiri, on the Nagpur–Jabalpur National Highway (NH 44), near Pench Tiger Reserve.

## 🚀 Hosting on GitHub Pages

1. Upload both files (`index.html` and `README.md`) to your GitHub repository.
2. Go to **Settings → Pages**.
3. Under **Source**, select `main` branch and `/ (root)` folder.
4. Click **Save** — your site will be live at `https://<your-username>.github.io/<repo-name>/`
